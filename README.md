# profile.fluid-line.ru

## Установка (dev)
- `cp .env.example .env`
- `composer install`
- `npm install`
- `npm run dev`

## Установка (prod)
- `cp .env.example .env`
- `composer install`
- `npm install`
- `npm run prod`
